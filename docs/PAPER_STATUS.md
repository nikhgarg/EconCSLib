# Public Paper Status

This page summarizes the paper formalizations currently published in the
public repository. Each paper folder contains the detailed theorem ledger,
source caveats, and review artifacts.

Machine-readable status sources are the paper-local
`papers/<PaperName>/status.json` files. The aggregate
[`papers/status.json`](../papers/status.json) is generated from those files with
`python3 scripts/sync_paper_status.py`.
Human-review counts are dashboard rows saved by a human reviewer; agent source
audits and stale ignored local logs are not counted.
