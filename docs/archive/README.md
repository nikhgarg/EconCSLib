# Archive

This folder holds historical plans and handoff notes that have been superseded
by current paper-folder `README.md` files or by the root repository status table.
Keep active paper status and handoff notes in the relevant
`papers/[Paper]/README.md`. Keep `FINAL_VALIDATION_REPORT.md` files in paper
folders only when they are final human-facing assessments of a paper or completed
proof phase, not routine handoff notes.

Current archive groups:

- `handoffs/`: old handoff notes superseded by current paper READMEs.
- `orientation/`: old human-orientation and paper-map docs with pre-`papers/`
  paths.
- `plans/`: historical planning docs.
- `status/`: replaced repository status snapshots.
