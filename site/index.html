EconCSLib: an AI-generated Lean 4 library for Economics and Computation

A Lean library and workflow for formalizing EC research papers.

EconCSLib is an experimental human-AI-Lean workflow to formalize EconCS papers: an LLM writes Lean code, Lean checks formal statements and proofs, and humans review the translation boundary from paper claims to formal statements.

Repository Paper

The project combines reusable EC Lean infrastructure with paper-specific formalization. The reusable library covers probability, optimization, matching, auctions, online algorithms, recommender systems, and fair division. The paper layer keeps one folder per source paper, preserving notation, theorem numbering, formal statements, proof code, and validation artifacts. The project also develops an LLM skill.md and formalization workflow to support AI-assisted formalization.

Contribute

Interested in contributing or formalizing your own paper?

Contributions are welcome while the contribution policy is still a draft. To protect unfinished work, new paper formalizations should start in a private workflow and be proposed through a public-safe pull request when ready. For questions or contributions, contact ngarg@cornell.edu.
